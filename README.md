# 商品バリデーションツール

CSVファイルをアップロードして商品データの重量・価格を自動チェックするツールです。

## 機能
- 重量上限のチェック
- 価格未設定の検知
- 結果のCSVダウンロード
- ブラウザで操作可能

## 使い方

### インストール
pip install -r requirements.txt

### 起動
python -m streamlit run app.py

ブラウザが自動で開きます。

## 操作手順
1. CSVファイルをアップロード
2. 重量上限をスライダーで設定
3. 実行ボタンを押す
4. 結果をCSVでダウンロード

## CSVフォーマット
| 品名 | 重さ | 値段 |
|------|------|------|
| 高級キーボード | 800 | 15000 |

## 使用ライブラリ
- streamlit
- pandas
- plyer
